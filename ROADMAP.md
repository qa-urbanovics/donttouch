# Red Trap — Roadmap (v1.0.1+)

## Gameplay
- [ ] Daily Challenge — один уровень в день для всех, борьба за лучший результат
- [ ] Fever Mode — после комбо x10 все зелёные дают x3 очков на 5 секунд
- [ ] Синяя плитка — двойной тап, даёт x3 очков
- [ ] Фиолетовая плитка — бомба, убирает все красные на экране
- [ ] Мигающая плитка — меняет цвет green↔red, нужно поймать момент
- [ ] Система жизней — 3 сердечка вместо мгновенного Game Over
- [ ] Boss Level — каждые 10 уровней специальный раунд

## Motivation & Retention
- [ ] Достижения — "Первые 100 очков", "Комбо x20", "Пережил Level 15", "50 игр"
- [ ] Streak система — играешь N дней подряд → разблокируй тему
- [ ] Таблица рекордов — история лучших 10 результатов
- [ ] Монеты — копишь за игры, покупаешь темы/скины

## Visual
- [ ] Темы оформления — Neon (текущая), Ocean, Sunset, Monochrome
- [ ] Анимация смерти — плитки разлетаются при Game Over
- [ ] Фоновая музыка — электронный бит, ускоряется с уровнем
- [ ] Скины плиток — круги, гексагоны, звёзды

## Social
- [ ] Game Center — глобальный лидерборд Apple
- [ ] Челлендж друзьям через iMessage
- [ ] Replay — запись последних 10 секунд, шеринг как видео

## Priority (first 3 for v1.0.1)
1. Достижения
2. Синяя плитка (двойной тап)
3. Game Center лидерборд
